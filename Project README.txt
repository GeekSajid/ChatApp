PRE REQUISITES:

ASP.NET Core SDK 3.1
Node.js 10.15.3 
Angular CLI 8.0
Visual Studio 2019
SQL Server 2014

Running The App:

1. After unzipping the code open the project using Visual Studio
2. From the package manager console run the following command- "Update-Database" it will create a new database for the project.
3. Run the application using visual studio.
